# Алгоритм расчёта витаминов и минералов

## Обзор

Данный калькулятор рассчитывает рекомендуемые суточные нормы потребления витаминов и минералов на основе данных USDA Dietary Reference Intakes (DRI).

## Источники данных

- **Источник 1**: [DRI Tables - NCBI](https://www.ncbi.nlm.nih.gov/books/NBK56056/)
- **Источник 2**: [DRI Tables - NCBI](https://www.ncbi.nlm.nih.gov/books/NBK222309/)

## Типы значений

### RDA (Recommended Dietary Allowance)
Рекомендуемая суточная норма потребления — средний дневной уровень потребления, достаточный для удовлетворения потребностей почти всех (97–98%) здоровых людей в определённой группе.

### AI (Adequate Intake)
Достаточное потребление — устанавливается, когда недостаточно данных для определения RDA. Предполагается, что AI обеспечивает достаточное питание.

### UL (Tolerable Upper Intake Level)
Верхний допустимый уровень потребления — максимальный дневной уровень потребления, который, вероятно, не представляет риска неблагоприятных последствий для здоровья.

## Группы населения

Калькулятор учитывает следующие группы:

| Код | Описание |
|-----|----------|
| 0-6 months | Младенцы 0-6 месяцев |
| 6-12 months | Младенцы 6-12 месяцев |
| 1-3 years | Дети 1-3 года |
| 4-8 years | Дети 4-8 лет |
| 9-13 years | Дети 9-13 лет |
| 14-18 years | Подростки 14-18 лет |
| 19-30 years | Взрослые 19-30 лет |
| 31-50 years | Взрослые 31-50 лет |
| 51-70 years | Взрослые 51-70 лет |
| 70+ years | Пожилые старше 70 лет |
| pregnancy 14-18 | Беременные (14-18 лет) |
| pregnancy 19-30 | Беременные (19-30 лет) |
| pregnancy 31-50 | Беременные (31-50 лет) |
| lactation 14-18 | Кормящие (14-18 лет) |
| lactation 19-30 | Кормящие (19-30 лет) |
| lactation 31-50 | Кормящие (31-50 лет) |

## Алгоритм определения группы

### Шаг 1: Определение базовой возрастной группы

На основе введённого возраста определяется базовая группа:
- 0-6 месяцев: возраст < 0.5
- 6-12 месяцев: возраст 0.5-1
- 1-3 года: возраст 1-3
- 4-8 лет: возраст 4-8
- 9-13 лет: возраст 9-13
- 14-18 лет: возраст 14-18
- 19-30 лет: возраст 19-30
- 31-50 лет: возраст 31-50
- 51-70 лет: возраст 51-70
- 70+ лет: возраст > 70

### Шаг 2: Учёт пола

Для некоторых минералов (железо, кальций) значения различаются для мужчин и женщин в определённых возрастных группах. При расчёте учитывается выбранный пол.

### Шаг 3: Выбор значения

1. Если для группы определён RDA — используется RDA
2. Если RDA не определён, но есть AI — используется AI
3. Если нет ни RDA, ни AI — значение не определено (отображается как "—")
4. UL отображается, если определён для группы

## Входные данные

Для расчёта используются:
- **Пол**: мужской / женский
- **Возраст**: число от 13 до 80 лет
- **Рост**: в сантиметрах
- **Вес**: в килограммах

## Ограничения

1. Калькулятор не учитывает беременность и лактацию (пользователь должен самостоятельно выбрать соответствующий режим)
2. Расчёт основан на общих рекомендациях DRI и не учитывает индивидуальные особенности
3. Значения для младенцев и детей младше 13 лет не рассчитываются (ограничение формулы Миффлина-Сан Жеора)

## Отображение результатов

Результаты отображаются в виде таблицы с тремя колонками:
1. **Название** — название витамина или минерала
2. **Норма (RDA/AI)** — рекомендуемая суточная норма
3. **Верхний предел (UL)** — верхний допустимый уровень

Если норма или верхний предел не определены, отображается прочерк "—".
