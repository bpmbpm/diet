# Словарь терминов

## Основные термины

### DRI (Dietary Reference Intakes)
**Диетические референсные потребления** — набор значений, используемых для планирования и оценки потребления питательных веществ здоровыми людьми. Включает RDA, AI, EAR и UL.

Подробнее: [Dietary Reference Intakes (DRI)](https://www.ncbi.nlm.nih.gov/books/NBK56068/)

### RDA (Recommended Dietary Allowance)
**Рекомендуемая суточная норма потребления** — средний дневной уровень потребления, достаточный для удовлетворения потребностей почти всех (97–98%) здоровых людей в определённой возрастной и гендерной группе.

Подробнее: [RDA Definition](https://www.ncbi.nlm.nih.gov/books/NBK56068/)

### AI (Adequate Intake)
**Достаточное потребление** — устанавливается, когда недостаточно научных данных для определения RDA. Предполагается, что AI обеспечивает достаточное питание для здоровых людей.

Подробнее: [AI Definition](https://www.ncbi.nlm.nih.gov/books/NBK56068/)

### UL (Tolerable Upper Intake Level)
**Верхний допустимый уровень потребления** — максимальный дневной уровень потребления питательного вещества, который, вероятно, не представляет риска неблагоприятных последствий для здоровья.

Подробнее: [UL Definition](https://www.ncbi.nlm.nih.gov/books/NBK56068/)

### EAR (Estimated Average Requirement)
**Средняя расчётная потребность** — средний дневной уровень потребления, который, как оценивается, удовлетворяет потребности 50% здоровых людей в определённой группе.

Подробнее: [EAR Definition](https://www.ncbi.nlm.nih.gov/books/NBK56068/)

## Термины калькулятора

### ИМТ (Индекс массы тела)
**Body Mass Index (BMI)** — показатель, используемый для оценки соответствия массы тела росту. Рассчитывается как отношение массы тела (кг) к квадрату роста (м²).

Подробнее: [ИМТ — Википедия](https://ru.wikipedia.org/wiki/Индекс_массы_тела)

### Базовая скорость метаболизма (BMR)
**Basal Metabolic Rate** — количество энергии (калорий), необходимое организму в состоянии покоя для поддержания жизненно важных функций.

Подробнее: [Basal metabolic rate — Wikipedia](https://en.wikipedia.org/wiki/Basal_metabolic_rate)

### Формула Миффлина-Сан Жеора
Уравнение для расчёта базовой скорости метаболизма, разработанное в 1990 году. Считается одной из наиболее точных формул для оценки потребности в калориях.

Подробнее: [Mifflin-St Jeor equation — Wikipedia](https://en.wikipedia.org/wiki/Basal_metabolic_rate#BMR_estimation_formulas)

## Витамины

### Витамин A
**Retinol** — жирорастворимый витамин, необходимый для зрения, иммунной функции и клеточного роста.

Подробнее: [Vitamin A — Wikipedia](https://en.wikipedia.org/wiki/Vitamin_A)

### Витамин C
**Аскорбиновая кислота** — водорастворимый витамин, антиоксидант, необходимый для синтеза коллагена и иммунной функции.

Подробнее: [Vitamin C — Wikipedia](https://en.wikipedia.org/wiki/Vitamin_C)

### Витамин D
**Кальциферол** — жирорастворимый витамин, необходимый для усвоения кальция и здоровья костей.

Подробнее: [Vitamin D — Wikipedia](https://en.wikipedia.org/wiki/Vitamin_D)

### Витамин E
**Токоферол** — жирорастворимый витамин, антиоксидант, защищает клеточные мембраны от окисления.

Подробнее: [Vitamin E — Wikipedia](https://en.wikipedia.org/wiki/Vitamin_E)

### Витамин K
**Филлохинон** — жирорастворимый витамин, необходимый для свёртывания крови и метаболизма костей.

Подробнее: [Vitamin K — Wikipedia](https://en.wikipedia.org/wiki/Vitamin_K)

### Витамин B1 (Тиамин)
**Thiamin** — водорастворимый витамин, необходимый для метаболизма углеводов и работы нервной системы.

Подробнее: [Thiamine — Wikipedia](https://en.wikipedia.org/wiki/Thiamine)

### Витамин B2 (Рибофлавин)
**Riboflavin** — водорастворимый витамин, участвует в энергетическом обмене и клеточном дыхании.

Подробнее: [Riboflavin — Wikipedia](https://en.wikipedia.org/wiki/Riboflavin)

### Витамин B3 (Ниацин)
**Niacin** — водорастворимый витамин, участвует в метаболизме и производстве энергии.

Подробнее: [Niacin — Wikipedia](https://en.wikipedia.org/wiki/Niacin)

### Витамин B6
**Pyridoxine** — водорастворимый витамин, необходимый для метаболизма аминокислот и гемоглобина.

Подробнее: [Vitamin B6 — Wikipedia](https://en.wikipedia.org/wiki/Vitamin_B6)

### Витамин B9 (Фолиевая кислота)
**Folate** — водорастворимый витамин, необходимый для синтеза ДНК и деления клеток.

Подробнее: [Folate — Wikipedia](https://en.wikipedia.org/wiki/Folate)

### Витамин B12
**Cobalamin** — водорастворимый витамин, необходимый для работы нервной системы и образования эритроцитов.

Подробнее: [Vitamin B12 — Wikipedia](https://en.wikipedia.org/wiki/Vitamin_B12)

## Минералы

### Железо (Iron)
Микроэлемент, необходимый для транспорта кислорода в крови (в составе гемоглобина) и многих ферментативных реакций.

Подробнее: [Iron — Wikipedia](https://en.wikipedia.org/wiki/Iron)

### Кальций (Calcium)
Макроэлемент, необходимый для здоровья костей, зубов, мышечной функции и передачи нервных импульсов.

Подробнее: [Calcium — Wikipedia](https://en.wikipedia.org/wiki/Calcium)

### Магний (Magnesium)
Макроэлемент, участвует в более чем 300 ферментативных реакциях, включая синтез белка и энергетический обмен.

Подробнее: [Magnesium — Wikipedia](https://en.wikipedia.org/wiki/Magnesium)

### Фосфор (Phosphorus)
Макроэлемент, необходимый для структуры костей, зубов, ДНК и АТФ.

Подробнее: [Phosphorus — Wikipedia](https://en.wikipedia.org/wiki/Phosphorus)

### Цинк (Zinc)
Микроэлемент, необходимый для иммунной функции, синтеза белка и заживления ран.

Подробнее: [Zinc — Wikipedia](https://en.wikipedia.org/wiki/Zinc)

### Йод (Iodine)
Микроэлемент, необходимый для синтеза гормонов щитовидной железы.

Подробнее: [Iodine — Wikipedia](https://en.wikipedia.org/wiki/Iodine)

### Селен (Selenium)
Микроэлемент с антиоксидантными свойствами, необходимый для иммунной функции и здоровья щитовидной железы.

Подробнее: [Selenium — Wikipedia](https://en.wikipedia.org/wiki/Selenium)

### Медь (Copper)
Микроэлемент, необходимый для метаболизма железа, формирования соединительной ткани и работы нервной системы.

Подробнее: [Copper — Wikipedia](https://en.wikipedia.org/wiki/Copper)

### Калий (Potassium)
Макроэлемент, необходимый для поддержания водного баланса, работы мышц и нервной системы.

Подробнее: [Potassium — Wikipedia](https://en.wikipedia.org/wiki/Potassium)

### Натрий (Sodium)
Макроэлемент, необходимый для поддержания водного баланса, работы мышц и нервной системы.

Подробнее: [Sodium — Wikipedia](https://en.wikipedia.org/wiki/Sodium)
