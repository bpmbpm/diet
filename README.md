# diet

## Сколько килокалорий нужно именно вам?!
Аксиома - “Даже в состоянии покоя (лежа на диване) человек тратит энергию”.   
По сути это и есть Основной обмен или BMR — Basal Metabolic Rate.  
Для офисного работника суточная калорийность складывается из BMR 80% и физической активности 20%.
